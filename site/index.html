aretext

Minimalist text editor that never slows you down.

Install

Key Features

  • Vim-compatible: edit efficiently with vim-compatible key bindings.
  • Searchable menus: quickly find files and commands.
  • Intuitive config: easily customize settings and menu commands for different projects.

Latest Release

Version 0.3 ("Russell") was released 2021-11-19.

This is a beta release, which means:

  • The user interface is stable, and I am using it as my primary editor.
  • Syntax highlighting is available for a few languages (Go, Python, C, Rust, YAML, JSON).
  • Many important vim commands are supported, but there are many more to add!
