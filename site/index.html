aretext

Minimalist text editor with vim-compatible key bindings.

install

Key Features

✓ Edit Efficiently

Supports nearly 100 of the most common vim commands in normal, insert, and visual modes. If you are comfortable with vim, you will feel right at home!

✓ Find Stuff Instantly

Built-in fuzzy search allows you to quickly find files and commands.

✓ Configure Confidently

Easily customize settings for different programming languages and projects.

Latest Release

Version 0.5 ("Heraclitus") was released 2022-05-08.

This is a beta release, which means:

  • The user interface is stable, and I am using it as my primary editor.
  • Syntax highlighting is available for several languages, including Go, Python, Rust, and C.
  • Many important vim commands are supported, but there are many more still to add!
