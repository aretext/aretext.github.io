Distraction Free

Simple and consistent terminal-based UI for distraction-free coding.

Vim Compatible

Supports over 100 vim commands in normal, insert, and visual modes. If you are comfortable with vim, you will feel right at home!

Customizable

Intuitive and flexible configuration, per language and project.

Terminal Native

Seamless integration with CLI tools. Create custom menu commands for tmux, git, code formatters, compilers, and anything else you use.
