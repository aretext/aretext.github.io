User Documentation

Welcome to the user documentation for aretext!

Aretext is a terminal-based text editor with vim-compatible key bindings.

If you are new to aretext, start here:

  • Install
  • Quickstart

These guides explain how to use aretext effectively:

  • Files: How to find, open, and save documents.
  • Navigation: How to navigate within a document.
  • Edit: How to edit a document.
  • Customization: How to customize the editor for your workflows.

For an overview of all available commands, see Command Reference

If you have questions about aretext, you can ask in GitHub Discussions.

If you want to report a bug or request a feature, please open a GitHub Issue.
